# JLPT N5 Essential Grammar Points

## Particles

| Particle | Usage | Example |
|----------|-------|---------|
| は (wa) | Topic marker | 私は学生です |
| が (ga) | Subject marker; existence sentences | 猫がいます |
| を (wo) | Direct object | 本を読みます |
| に (ni) | Direction / destination / time / indirect object / location of existence | 学校に行く / 3時に起きる / 机に本がある |
| で (de) | Location of action / means / material | 公園で遊ぶ / バスで来る |
| の (no) | Possession / noun modifier | 私の本 / 日本語の先生 |
| と (to) | Together with / listing | 友達と行く / りんごとみかん |
| も (mo) | Also / too | 私も行きます |
| か (ka) | Question marker | これは何ですか |
| へ (e) | Direction (more literary than に) | 東京へ行く |
| から (kara) | From (start point) / because | 9時から / 寒いから着る |
| まで (made) | Until / up to | 5時まで / 駅まで |
| より (yori) | Comparison (than) | 兄より背が高い |
| だけ (dake) | Only / just | 一つだけ |
| ね (ne) | Seeking agreement | いい天気ですね |
| よ (yo) | Emphasis / informing | そうですよ |

## Verb conjugations (polite)

| Form | Usage | Example |
|------|-------|---------|
| ～ます | Non-past affirmative | 食べます |
| ～ません | Non-past negative | 食べません |
| ～ました | Past affirmative | 食べました |
| ～ませんでした | Past negative | 食べませんでした |
| ～ましょう | Volitional / suggestion | 行きましょう |
| ～てください | Request | 見てください |
| ～ています | Progressive / state | 読んでいます |

| Type | Non-past | Past | Negative |
|------|----------|------|----------|
| い-adj | 高い | 高かった | 高くない |
| な-adj | 静かです | 静かでした | 静かじゃない |

## Key sentence patterns

| Pattern | Meaning | Example |
|---------|---------|---------|
| ～がある / ～がいる | There is (thing/person) | 机の上に本があります |
| ～たい | Want to | 食べたいです |
| ～てもいい | May / allowed to | 帰ってもいいです |
| ～なければならない / ～なきゃ | Must | 行かなければなりません |
| ～から | Because | 雨が降っているから、傘を持ちます |
| ～と思う | I think that | いい映画だと思います |
| ～ませんか | Won't you? / Invitation | 一緒に行きませんか |
| ～でしょう | Probably | 明日は晴れるでしょう |
| AよりB (の方が)～ | B is more ~ than A | 電車より地下鉄の方が速い |
| ～ながら | While doing | 音楽を聴きながら勉強する |

## Common distractor pairs (particles)

- **に vs で**: destination/existence (に) vs. action location/means (で)
- **は vs が**: topic (は) vs. subject/new info (が)
- **を vs に**: direct object (を) vs. indirect object/direction (に)
- **と vs に**: accompaniment (と) vs. direction/indirect (に)
- **から vs で**: starting point (から) vs. material/means (で)
