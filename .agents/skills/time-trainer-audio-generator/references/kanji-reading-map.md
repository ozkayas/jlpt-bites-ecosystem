## Known TTS Issues

| Problem | Description | Solution |
|---------|-------------|----------|
| `8分` alternative | TTS sometimes reads `はちふん` instead of `はっぷん` | Both accepted in verification |
| `12分/13分` repeat | TTS occasionally repeats segments | Retry triggers new batch generation |
| `0時` reading | TTS may say `ぜろじ` or `れいじ` | Both accepted |
