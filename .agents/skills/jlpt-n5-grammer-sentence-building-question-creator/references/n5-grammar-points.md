# N5 Grammar Points for Sentence Building

## Core Sentence Patterns (SOV)

Japanese follows Subject–Object–Verb order. Sentence building questions test whether students can reconstruct this order from scrambled parts.

## Key Word-Order Rules

| Rule | Pattern | Example |
|------|---------|---------|
| Topic first | Xは ... | 私は毎日勉強します |
| Location + で before verb | 場所で + V | 公園で遊びます |
| Destination + に/へ before movement verb | 場所に + 行く/来る | 学校に行きます |
| Object + を before verb | Nを + V | 本を読みます |
| Time before location | 時間 + 場所で + V | 毎朝公園で走ります |
| Accompaniment early | Nと一緒に | 友達と一緒に行く |
| Modifier before noun | AdjN / NのN | きれいな花 / 日本の本 |
| て-form chaining | V1て + V2 | 起きてご飯を食べる |

## Common Scramble Combinations

### Particle + Noun pairs
- 場所 + で/に/へ (location)
- 名詞 + を (object)
- 名詞 + が (subject)
- 名詞 + の + 名詞 (possession)
- 名詞 + と (accompaniment/listing)

### Verb-ending patterns
- V + ています (progressive)
- V + たいです (want to)
- V + てください (request)
- V + ましょう (suggestion)
- V + ませんか (invitation)

### Adjective patterns
- い-adj + 名詞 (大きい犬)
- な-adj + な + 名詞 (きれいな花)

## Difficulty Guidelines

- Use 2 particles + 2 content words for basic questions
- Mix a grammar pattern with particles for intermediate questions
- Ensure only one correct word order exists
- The reconstructed sentence must sound natural
