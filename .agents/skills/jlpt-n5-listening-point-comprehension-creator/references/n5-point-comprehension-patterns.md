# N5 Point Comprehension (Mondai 2) — Pattern Reference

This document contains real question patterns extracted from 30 JLPT N5 listening questions.
Use these as blueprints when creating variations.

---

## Pattern 1: Fikir Degistirme (Change of Mind)

### Template
```
Speaker A proposes Option X.
Speaker B raises a concern / suggests Option Y.
Speaker A (or B) agrees to Option Y (or a third Option Z).
```

### Real Examples

**Birthday Present (Hediye):**
- Teklif 1: とけい (saat) -> pahalı -> RED
- Teklif 2: おさけ (alkol) -> sevmiyor -> RED
- Teklif 3: おさいふ ya da ねくたい -> kravat istiyordu -> KABUL
- Correct: ねくたい (kravat)
- Distractors: とけい, おさけ, おさいふ

**Meeting Place (Bulusma Yeri):**
- Teklif 1: えき (istasyon) -> KABUL gibi gorunuyor
- Teklif 2: でぱーと icinde yemek -> "それがいいですね" -> KABUL
- Correct: でぱーと
- Distractors: えき, れすとらん, えいがかん

**Drinks (Icecek):**
- Istek: こーひー -> yok -> RED
- Istek 2: あたたかいもの -> sıcak çay yapılabilir -> KABUL
- Correct: あたたかいおちゃ
- Distractors: こーひー, びーる, じゅーす

### Option Design for This Pattern
- Option A: First proposal (rejected)
- Option B: Second proposal (if rejected, this too)
- Option C: CORRECT (final decision)
- Option D: Something mentioned in dialogue but never proposed

## Pattern 2: Dikkat Dagitici (Distractor Info)

### Template
```
Multiple people describe their plans/situations.
Question asks about ONE specific person.
```

### Real Examples

**Summer Vacation:**
- Kadın: アルバイト (part-time job)
- Kadın: りょこう (travel - istediği ama yapamadığı)
- Erkek: やまにのぼる (mountain climbing) <- SORU BU
- Erkek'in ebeveynleri: はしる (running - hazırlık)
- Correct: やまにのぼります
- Distractors: アルバイトをします, りょこうをします, まいにちはしります

**Siblings' Jobs:**
- Ağabey: びょういん (hospital)
- Erkek kardeş: でんきのかいしゃ (electric company)
- Kız kardeş: にほんごをおしえる (teach Japanese) <- SORU BU
- Kadının ablası: ぎんこう (bank)
- Correct: がいこくじんににほんごをおしえています

### Option Design for This Pattern
- Each option = a different person's plan/situation
- Only ONE matches the person asked about

## Pattern 3: Duzeltme (Correction)

### Template
```
Speaker lists items: A, B, C.
Speaker immediately corrects: "Actually C is not there, only A and B."
```

### Real Example

**Forgotten Bag:**
- "かぎと てがみと ぺんが はいっています"
- "あ、ぺんは ぽけっとに はいっていますから かぎと てがみだけです"
- Correct: かぎとてがみ (keys and letter)
- Distractors: bags with pen included, bags with wrong items

### Option Design for This Pattern
- Option with corrected info: CORRECT
- Option with uncorrected info: DISTRACTOR
- Option with partial info: DISTRACTOR
- Option with extra wrong info: DISTRACTOR

## Pattern 4: Hesaplama (Calculation)

### Template
```
Numbers mentioned in dialogue.
Listener must combine them to get the answer.
```

### Real Examples

**Tomato Price:**
- Tanesi 80 yen, 3 tane istiyor
- Satıcı: 2 tane 150 yen
- Müşteri: "bir tane daha" = 4 tane = 2 set = 300 yen
- Correct: 300 yen
- Distractors: 240 (3x80), 150 (1 set), 230 (3x80-10 gibi yanlış hesap)

**Copy Count:**
- 45 öğrenci + 5 öğretmen = 50 kişi
- 2 kopya/kişi = 100 kopya
- Correct: 100
- Distractors: 50, 90 (45x2), 45

**Arrival Time:**
- Normal: 8:45
- Bugün: "1 saat erken" = 7:45
- Correct: 7:45
- Distractors: 8:45, 8:30, 8:00

### Option Design for This Pattern
- Options are numbers/times/prices
- Correct = full calculation
- Distractors = partial calculations or raw numbers

## Pattern 5: Zaman Cikarimi (Time Inference)

### Template
```
A reference date/time is given.
A relative expression points to the actual answer.
```

### Real Examples

**Birthday 11/11:**
- "おとといです。じゅういちにちです。"
- "いちがよっつならびますね" = 11月11日
- Correct: 11月11日

**Hospital Appointment:**
- Bugün: Pazartesi
- İş: Çarşamba'ya kadar
- Doktor: "つぎのひは?" = Perşembe
- Correct: Perşembe öğleden sonra 2'de

**Homework Return:**
- Teslim: Pazartesi
- İade: "つぎのひ" = Salı
- Correct: Salı

### Option Design for This Pattern
- Options are dates/days/times
- Correct = the inferred time
- Distractors = reference time, wrong offsets, other mentioned times

## Pattern 6: Olumsuz Eleme (Negative Elimination)

### Template
```
Speaker lists things that are NOT available / NOT allowed.
What remains is the answer.
```

### Real Examples

**Museum Items:**
- Getirin: おかね, ぺん, のーと
- Getirmeyin: かめら (しゃしんをとってはいけません), たべもの, のみもの
- Correct: おかねとぺんとのーと

**Hospital Room:**
- Odada yok: といれ, しゃわーるーむ
- Odada var: ちいさいれいぞうこ
- Correct: れいぞうこ

### Option Design for This Pattern
- Correct = what IS available/allowed
- Distractors = things that are NOT available/forbidden

---

## Pattern 7: Siparis/Secim (Selection)

### Template
```
Multiple options offered.
Customer asks for specific item, gets it or settles for alternative.
```

### Real Example

**Restaurant Ordering:**
- Menu: ぶたにく, さかな, とりにく
- Müşteri: "とりにくはありませんか?"
- Garson: "はい、こちらです"
- Müşteri: "これをおねがいします"
- Correct: とりにく

### Option Design for This Pattern
- Correct = final selection
- Distractors = other menu items / rejected options

---

## Pattern 8: Neden/Sebep (Reason)

### Template
```
Question asks WHY something happened.
Multiple possible reasons mentioned, only one is actual.
```

### Real Example

**Late for Class:**
- Öğretmen soruyor: hasta mı? -> "いいえ、げんきです"
- Gerçek sebep: Yağmur -> bisiklet yerine tren -> istasyondan yolu bilmiyor
- Correct: えきからがっこうまでのみちがわかりませんでした
- Distractors: おなかがいたい, あめがふった, でんしゃがおくれた

### Option Design for This Pattern
- Correct = the actual reason stated
- Distractors = suggested but denied reasons, contributing factors that aren't the main reason
