# JLPT N5 Allowed Grammar Points

Use ONLY the grammar points below when creating N5 listening questions. If a grammar point is not on this list, do NOT use it.

## Particles
| Particle | Usage | Example |
|----------|-------|---------|
| は | Topic marker | わたし**は** がくせいです。 |
| が | Subject marker | ねこ**が** います。 |
| を | Object marker | パン**を** たべます。 |
| に | Direction / Time / Location | がっこう**に** いきます。 / 7じ**に** おきます。 |
| で | Location of action / Means | こうえん**で** あそびます。 / バス**で** いきます。 |
| へ | Direction | うみ**へ** いきます。 |
| と | With / And (nouns) | ともだち**と** いきます。 |
| も | Also | わたし**も** いきます。 |
| の | Possession / Noun modifier | わたし**の** ほん。 |
| から | From (time/place) | 9じ**から** べんきょうします。 |
| まで | Until (time/place) | 5じ**まで** はたらきます。 |
| や | And (non-exhaustive list) | りんご**や** みかん |
| か | Question marker | これは なんです**か**。 |
| ね | Confirmation tag | いいてんきです**ね**。 |
| よ | Emphasis / New info | おいしいです**よ**。 |

## Verb Forms
| Form | Usage | Example |
|------|-------|---------|
| ます / ません | Polite present/negative | たべ**ます** / たべ**ません** |
| ました / ませんでした | Polite past/negative past | たべ**ました** / たべ**ませんでした** |
| ましょう | Let's / Shall we | いき**ましょう**。 |
| ませんか | Invitation | いっしょに いき**ませんか**。 |
| て form | Connecting / Request | たべ**て**ください。 |
| ている | Ongoing action / State | いま たべ**ています**。 |
| たい | Want to | たべ**たい**です。 |
| ないでください | Please don't | ここで たばこを すわ**ないでください**。 |

## Adjective Forms
| Form | Usage | Example |
|------|-------|---------|
| い-adjective | Direct modifier | **おおきい** いぬ |
| い → くない | Negative | おおき**くない**です。 |
| い → かった | Past | おおき**かった**です。 |
| な-adjective + な | Modifier | **しずかな** まち |
| な-adjective + じゃない | Negative | しずか**じゃない**です。 |
| な-adjective + でした | Past | しずか**でした**。 |

## Sentence Patterns
| Pattern | Meaning | Example |
|---------|---------|---------|
| AはBです | A is B | これは ほんです。 |
| AはBじゃないです | A is not B | これは ほんじゃないです。 |
| AにBがあります/います | There is B at A | つくえの うえに ほんが あります。 |
| AはBより〜 | A is more ~ than B | とうきょうは おおさかより おおきいです。 |
| AとBとどちらが〜 | Which is more ~, A or B? | にくと さかなと どちらが すきですか。 |
| 〜のなかで〜がいちばん | Among ~, ~ is the most | くだもののなかで りんごが いちばん すきです。 |
| 〜たことがあります | Have experienced | にほんに いったことが あります。 |
| 〜まえに | Before ~ | たべるまえに てを あらいます。 |
| 〜あとで | After ~ | たべたあとで さんぽします。 |
| 〜とき | When ~ | ひまなとき ほんを よみます。 |
| 〜から (reason) | Because ~ | あついですから まどを あけましょう。 |
| 〜けど / が | But | いきたいですけど じかんが ありません。 |
| 〜てもいいですか | May I ~? | ここに すわってもいいですか。 |
| 〜てはいけません | Must not ~ | ここで しゃしんを とってはいけません。 |
| 〜なければなりません | Must ~ | しゅくだいを しなければなりません。 |
| 〜ほうがいい | Better to ~ | はやく ねたほうがいいですよ。 |
| 〜つもりです | Plan to ~ | あした えいがを みるつもりです。 |
| 〜でしょう | Probably | あしたは あめでしょう。 |
| 〜と おもいます | I think ~ | おいしいと おもいます。 |

## Counters (Common)
| Counter | For |
|---------|-----|
| 〜にん | People (ひとり、ふたり、さんにん...) |
| 〜つ | General objects (ひとつ、ふたつ...) |
| 〜まい | Flat objects (sheets, tickets) |
| 〜ほん/ぽん/ぼん | Long/cylindrical objects |
| 〜さつ | Books |
| 〜だい | Machines, vehicles |
| 〜ひき/ぴき/びき | Small animals |
| 〜かい | Floors |
| 〜じ | Hours (time) |
| 〜ふん/ぷん | Minutes |
